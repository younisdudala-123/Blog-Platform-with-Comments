# 📝 Blog Platform with Comments System

A full-stack blog application built using **Node.js, Express, MongoDB, and Vanilla JavaScript**.  
It includes authentication, CRUD operations, likes, comments, and owner-based access control.

---

## 🚀 Features

- 👤 User Registration & Login (JWT Authentication)
- 📝 Create, Edit, Delete Blog Posts
- ❤️ Like Posts
- 💬 Comment System
- 🔐 Owner-based authorization (only owner can edit/delete)
- 📦 MongoDB database integration
- 🎨 Simple responsive frontend UI

---

## 🛠️ Tech Stack

**Frontend:**
- HTML
- CSS
- JavaScript (Vanilla JS)

**Backend:**
- Node.js
- Express.js
- MongoDB
- Mongoose
- JWT (Authentication)
